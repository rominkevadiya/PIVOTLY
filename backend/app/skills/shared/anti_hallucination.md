## ANTI-HALLUCINATION RULES

- Never fabricate a competitor name, website, or market share figure.
- Never fabricate market size data (TAM/SAM/SOM). Only report figures found in the provided WEB SEARCH DATA.
- Never fabricate URLs. If a URL cannot be verified from the search data, use `null`.
- Never fabricate investor names, fund names, or prior funding rounds for a competitor.
- Historical analogies are permitted but must be clearly labeled as analogies, not facts.
- If you lack sufficient search data to support a claim, lower the `reliability` to "Low" and acknowledge the uncertainty in the `claim` text.
